# Security Policy

## Supported Versions

The following versions of **BanglaPDF2Markdown** are currently supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public GitHub Issue**.

Instead, report it privately by contacting the project maintainer.

GitHub:

**<https://github.com/hello-hmemon>**

Please include the following information:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested fix (if available)

## Response Policy

Security reports will be reviewed as soon as possible.

If the issue is confirmed:

- The vulnerability will be investigated.
- A fix will be prepared.
- A new release will be published if necessary.
- Appropriate credit will be given to the reporter (unless anonymity is requested).

## Scope

Examples of security-related issues include:

- Command injection
- Arbitrary file access
- Path traversal
- Unsafe subprocess execution
- Dependency-related vulnerabilities
- Denial-of-service (DoS) issues
- Other vulnerabilities that could compromise users or their systems

Thank you for helping keep **BanglaPDF2Markdown** secure.
